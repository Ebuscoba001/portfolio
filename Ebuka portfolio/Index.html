HTML & CSS
HTML is the standard markup language for creating and designing web pages. It structures the content on the web and uses tags to define elements such as headings, paragraphs, links, images, and more. HTML provides the basic building blocks for creating a web page and is often used in conjunction with CSS and JavaScript.

JavaScript
JavaScript is a dynamic programming language primarily used for client-side scripting in web development. It enables interactivity on web pages, allowing developers to create dynamic content, handle user input, and modify the DOM (Document Object Model) in real-time. JavaScript is widely supported by web browsers and is a fundamental technology for building modern web applications.
Link text

PHP fullstack

PHP is a server-side scripting language designed for web development. It is embedded in HTML code and executed on the server, generating dynamic content before being sent to the client's browser. PHP is commonly used for tasks such as processing form data, interacting with databases, and creating dynamic web pages. It is a versatile language that powers many websites and web applications.
Link text

Python

Python is a high-level, general-purpose programming language known for its readability and versatility. In the context of web development, Python is often used for server-side scripting, creating web applications, and building APIs (Application Programming Interfaces). Popular web frameworks like Django and Flask leverage Python to simplify the development of robust and scalable web applications.
